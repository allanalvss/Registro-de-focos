# Registro-de-focos
O Registro de focos tem como principal objetivo cadastrar os bairros da cidade e os focos de dengue encontrados em cada um deles.A figura abaixo apresenta a tela inicial da aplicação.


![](figuras/inicial.png)


Ademais, o aplicação apresenta uma janela para estatísticas, na qual são apresentados a "Média de focos" encontrados e consequentemente os bairros classisificados com maior e menor risco.

![](figuras/estat.png)

# Inserção dos dados

Os dados necessários para a aplicação são o nome do bairro e número de focos registrados naquele local. É necessário preencher os labels (Bairro e Quantidade de Focos).E logo após isso o botão inserir irá atualializar a tabela e classificar o nível de risco de cada bairro cadastrado.

![](figuras/label.png)
